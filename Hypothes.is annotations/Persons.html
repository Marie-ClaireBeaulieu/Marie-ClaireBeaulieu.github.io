Persons
Identify the people in the social network

  1. Annotate a person mentioned in the Smith's entry with the Stable Identifier for that person
  2. Highlight a person name to initiate an annotation
  3. Locate the stable id of the person resource and add it to the annotation
  4. Add keywords:
    1. person to identify this as a person annotation
    2. PSN to identify the annotation was created for "Perseids Social Network"
  5. Set annotation to public
  6. Save
