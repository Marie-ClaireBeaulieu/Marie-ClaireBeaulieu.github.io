Prerequisites
What you need to do before getting started

  1. Before starting, make sure you have an updated version of either the Chrome or Firefox web browser
  2. Create a single Perseids account for your team. Use one of the team member's Tufts email account or create a dedicated gmail account for this purpose.
  3. List all the full names of teammates in the full name field
  4. Enter a nickname for your group. Remember it, this is how I will be able to identify you when grading
  5. In the Affiliation field, write “Tufts Myth 2016”

This process allows us to identify annotators and give you proper credit for your work
