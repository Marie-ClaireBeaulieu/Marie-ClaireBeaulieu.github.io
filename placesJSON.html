Instructions for Place Annotations Using JSON-LD

Step 1

Copy and paste the following in a plain text document:

      {
          "@context": "http://www.w3.org/ns/oa-context-20130208.json",
            "@id": " Glaucus",   
            "annotatedBy": {
               "@type": "foaf:Person",  
                "@id": "http://sosol.perseids.org/sosol/users/Marie-Claire%20Beaulieu"   
            },
            "@type": "oa:Annotation",  
            "dcterms:title": "Identifies Potniae as the town in which Glaucus gets devoured by his mares",  
            "annotatedAt": "2015-12-01",  
            "motivatedBy": "oa:identifying",  
        
            },  
            "hasTarget": {  
            "@id": "Potniae-Glaucus#target-1",  
            "@type": "oa:SpecificResource",  
            "hasSelector": {  
                "@id": "Potniae-Glaucus#target-1-sel-1",  
                "@type": "oa:TextQuoteSelector",  
                "exact": "Potniae",  
                "prefix": "he lived at",   
                "suffix": ", despised the power of Aphrodite"  
               },  
            "hasSource": {  
              "@id": "urn:cts:pdlrefwk:viaf88890045.003.perseus-eng1:G.glaucus_1"   
               }  
            },  
            "hasBody": [  
            {  
               "@id": "http://pleiades.stoa.org/places/541070#this"  
]

Step 2

Customize with the appropriate information for the annotation you are creating, as follows

      
        {
          "@context": "http://www.w3.org/ns/oa-context-20130208.json", leave as is
            "@id": "Potniae Glaucus",   create a short ID for your annotation. Each must be unique
            "annotatedBy": { leave as is
               "@type": "foaf:Person",  leave as is
               "@id": "http://sosol.perseids.org/sosol/users/Marie-Claire%20Beaulieu" "http://sosol.perseids.org/sosol/users/balmas01@tufts.edu"Find this ID on Perseids
            },
            "@type": "oa:Annotation",  leave as is
            "dcterms:title": "Identifies Potniae as the town in which Glaucus gets devoured by his mares",  create a descriptive ID for your annotation
            "annotatedAt": "2015-12-01",  year-month-date
            "motivatedBy": "oa:identifying",   leave as is
        
            },  
            "hasTarget": {  leave as is
            "@id": "Potniae-Glaucus#target-1",  reuse the short ID you have created, followed with #target-1. If you have only one target (which is the case for this project), all your annotations will have #target-1
            "@type": "oa:SpecificResource",  leave as is
            "hasSelector": {  leave as is
                "@id": "Potniae-Glaucus#target-1-sel-1",  reuse the short ID you have created, followed with #target-1-sel-1 If you have only one target and one selector (which is the case for this project), all your annotations will have #target-1-sel-1
                "@type": "oa:TextQuoteSelector",  leave as is
                "exact": "Potniae",  enter the place name that constitutes your annotation
                "prefix": "he lived at",   enter the few words that precede the place name in the Smith entry
                "suffix": ", despised the power of Aphrodite"  enter the few words that follow the place name in the Smith entry
               },  
            "hasSource": {  leave as is
              "@id": "urn:cts:pdlrefwk:viaf88890045.003.perseus-eng1:G.glaucus_1"   Customize the initial and name ID for your entry with the name of the mythological figure you are annotating 
               }  
            },  
            "hasBody": [  
            {  
               "@id": "http://pleiades.stoa.org/places/541070#this"  Find the place name on Pleiades, copy and paste the url here, and add "#this" at the end in order to indicate that this is a specific resource
